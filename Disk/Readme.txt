Данный документ содержит краткое описание программной части дипломного проекта:
"Личный кабинет сотрудника БГУИР".

Целью данного проекта является реализация личного кабинета сотрудника БГУИР.
При создании данного проекта использовались технологии:
- язык программирования Java, библиотека Spring;
- JavaScript, библиотека React;
- спецификация Java EE - Java Persistence API (JPA);
- Maven, для сборки сервера.

При реализации веб-приложения используется реляционная база данных PostgreSQL.

Для развертывания данного приложения необходимо:
- запустить исполняемый файл Epa.exe, который разархивирует полный проект
- после распаковки архива для разворачивания веб-приложения необходимо пройти шаги, указанные в руководстве пользователя.
Также для прохождения инструкции, указанной в руководстве пользователя, необходимо стабильное подключение к сети Интернет.
